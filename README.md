# Voice to image

A very simple flask app to convert voice to images.

## Usage

### Dependencies

- python 3.10
- pip

### Installation

```bash
git clone https://github.com/sud0cs/voice-to-image
cd voice-to-image
python3.10 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Initialization

```bash
python3.10 -m venv venv
source venv/bin/activate
python3 server.py
```
